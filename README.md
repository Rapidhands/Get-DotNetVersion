# Get-DotNetVersion

Retrieve .NET Framework version from one or more computers

## Getting Started

Run

### Prerequisites

Script requires PowerShell 3.0 and higher.

### Usage

.\Get-DotNetVersions.ps1 -ComputerName ex1, ex2
